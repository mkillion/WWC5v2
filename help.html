General Info

  • The mapper is now mobile friendly, optimized for use on desktop computers, tablets, and most phones. To go directly to the mapper without going through the KGS website, bookmark this link: WWC5 Mapper
  • Settings such as layer visibility, labeling, classification, bookmarks, and last location are now saved between sessions by default. To completely clear these settings, clear the browser's history, including cookies.
  • The map can now be printed directly from the browser's Print command - use this option to include the well information popup. Or use the Print / Save Map tool to save the map to a PDF or other image format, with title and other map elements.
  • In addition to the zoom buttons, you can zoom in by holding the Shift key and drawing a box with the mouse.

Tools Panel

Buffer / Radius
  • Creates a buffer of a specified radius around a selected well, user-defined point, or searched address.
  • Highlight a well by selecting it, or double-click on a desired spot on the map to define a point, before using the tool.
  • Enter a buffer distance and select units.
  • Checking the "Select wells inside buffer" checkbox will display only the wells inside the buffer, and only those wells will be included in a data download.
  • Click on the eraser icon at the top of the panel to erase graphics and clear all buffers and filters.
Classify Wells
  • Select a variable on which to color-code wells. Click on the Links/Legend panel to view the legend.
Download Data
  • Creates comma-delimited text files with well and lithologic log information for wells visible in the current map extent.
  • Only wells visible on the map will be included in the download files. Applying a filter will restrict which wells are included.
  • For other options to download well data visit the WWC5 Database.
  • To download a shapefile of all WWC5 wells, visit the DASC catalog.
Draw
  • Select the appropriate tool to draw point, line, rectangle, circle, and freehand polygon graphics on the map.
  • When drawing freehand polygons, either hold the mouse down and draw, or click individual points to shape the polygon.
  • Polygons can be reshaped or moved by selecting the Move or Reshape tool, then clicking on the polygon to move it or clicking on a vertex to reshape it.
  • Checking the "Select wells" box before drawing will select wells within the graphic and only those wells will be included in a data download.
  • The drawing graphic and selected wells can be cleared by clicking the Clear button in the Draw tool or the eraser icon at the top of the panel.
Filter Wells
  • Restrict which wells are visible on the map (and included in downloaded text files) by applying a filter.
  • Filters are cumulative - e.g. "wells that are domestic AND were completed after 01/23/2015."
  • Click the Reset button to clear the filter and reset all input fields. Clicking the eraser icon at the top of the panel will also clear the filter.
Label Wells
  • Select a variable to label well symbols.
Measure Area and Distance
  • Select "New Measurement" from either the Area or Distance section to begin a measurement.
  • Draw on the map, then double-click to close the polygon or end the line.
  • Change units by selecting from the Units dropdown box.
  • Clear the measurement and graphic by clicking the Clear Area/Distance Measurement buttons in the Measure tool or the eraser icon at the top of the tools panel.
Print / Save Map
  • Enter a title for the map (optional).
  • Select page layout and size.
  • Select output format.
  • Checking the "Include legend" checkbox will put a legend of wells symbols at the bottom of the printout.
  • When the export is finished (may take up to 60 seconds or so), a download link will appear beneath the "Print / Save Map" button.
  • The map (map image only) can also be printed using the browser's Print command.

Map Window
